# AION Quantum Supreme v6.0

Prepared package for GitHub/Render deployment.

## Setup
1. Create repo and add these files.
2. Create `.env` from `.env.example` and fill your Binance TESTNET keys, set MODE=PAPER.
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python main.py

**Important:** Start with MODE=PAPER (testnet) until you validate behavior.
